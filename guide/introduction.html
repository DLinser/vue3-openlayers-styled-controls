介绍 ​

Vue3 OpenLayers Styled Controls 是一套基于 Vue 3 和 OpenLayers 的可定制样式地图控件库。该库专注于为 Web 地图应用提供美观、响应式的 UI 控件组件。

特性 ​

  • Vue 3 Composition API: 基于 Vue 3 的 <script setup> 语法糖开发
  • TypeScript 支持: 完整的 TypeScript 类型定义
  • 可定制样式: 支持 SCSS 变量定制和多种主题
  • 国际化支持: 内置多语言支持，易于扩展
  • 轻量级: 专注于核心功能，保持组件库轻量高效
  • 易于集成: 简单的 API 设计，快速集成到现有项目中

支持的控件 ​

  • OlStyledControlBar: 控件容器，用于水平排列多个控件
  • OlStyledZoomInControl: 自定义样式化的放大控件
  • OlStyledZoomOutControl: 自定义样式化的缩小控件
  • OlStyledFullScreenControl: 全屏切换控件
  • OlStyledMeasureLengthControl: 长度测量控件
  • OlStyledMeasureAreaControl: 面积测量控件
  • OlStyledClearControl: 清除测量结果控件
  • OlStyledSidebarControl: 侧边栏切换控件

浏览器支持 ​

  • Chrome (最新版)
  • Firefox (最新版)
  • Safari (最新版)
  • Edge (最新版)

技术栈 ​

  • Vue 3 (^3.0.0)
  • TypeScript (~5.9.3)
  • OpenLayers (^10.7.0)
  • vue3-openlayers (^11.6.2)
  • Vite (^7.1.7)
