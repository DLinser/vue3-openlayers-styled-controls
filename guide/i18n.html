国际化 ​

Vue3 OpenLayers Styled Controls 支持国际化，可以轻松适配不同语言环境。

默认语言 ​

组件库默认提供以下语言支持：

  • 简体中文 (zh-CN)
  • 英文 (en)

使用方式 ​

1. 安装 vue-i18n ​

bash
npm install vue-i18n@latest

2. 配置 i18n ​

js
// main.js
import { createApp } from 'vue'
import { createI18n } from 'vue-i18n'
import App from './App.vue'

// 导入组件库的 i18n 配置
import { createComponentI18n } from 'vue3-openlayers-styled-controls/i18n'

const i18n = createComponentI18n({
  locale: 'zh-CN', // 设置默认语言
  fallbackLocale: 'en' // 设置回退语言
})

const app = createApp(App)
app.use(i18n)
app.mount('#app')

3. 自定义语言资源 ​

如果您需要自定义语言资源或添加更多语言：

js
const i18n = createComponentI18n({
  locale: 'en',
  messages: {
    en: {
      zoomInTitle: 'Zoom In',
      zoomOutTitle: 'Zoom Out',
      fullScreenTitle: 'Toggle Full Screen',
      measureLengthTitle: 'Measure Length',
      measureAreaTitle: 'Measure Area',
      clearTitle: 'Clear Measurements',
      sidebarTitle: 'Toggle Sidebar'
    },
    zh-CN: {
      zoomInTitle: '放大',
      zoomOutTitle: '缩小',
      fullScreenTitle: '全屏切换',
      measureLengthTitle: '测量长度',
      measureAreaTitle: '测量面积',
      clearTitle: '清除测量',
      sidebarTitle: '侧边栏切换'
    }
  }
})

在组件中使用 ​

组件库中的控件会自动使用 i18n 配置：

vue
<template>
  <!-- 控件会自动显示对应语言的 title -->
  <OlStyledZoomInControl />
  <OlStyledZoomOutControl />
</template>

扩展语言支持 ​

如果您需要添加更多语言支持，可以通过以下方式：

js
const i18n = createComponentI18n({
  locale: 'ja',
  messages: {
    ja: {
      zoomInTitle: '拡大',
      zoomOutTitle: '縮小',
      // ... 其他翻译
    }
  }
})
